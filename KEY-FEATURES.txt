
• List Key Features:
I.	We will have a dental chart from which the doctor can select each tooth and relate the diagnosis to that specific tooth.
II.	The doctor will be able to see from the dental chart the condition of each tooth and if it was treated before.
III.	The system will include as much shortcuts as possible to help make it easier for the doctor working with the system.
IV.	The doctor will have a calendar from which he can check all his current and upcoming appointments.
V.	The system will have a tab for the treatment plan which will be set by the doctor based on the diagnoses and future visits.
VI.	Notification system which sends reminders before appointments to patients.
VII.	We will have the database of the clinic’s inventory to check the availability of the supplies in that clinic.
VIII.	The ability to get inventory and sales reports from a given date.
